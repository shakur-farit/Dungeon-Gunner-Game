# Dungeon-Gunner-Game
Rouglike dungeon shooter game.

Implemented mechanics:
 1. Generate randome dungeon.
 2. Player movement.
 3. Player aiming.
 4. Player animation.
 5. Player shooting.
 6. Weapons changing and reloading.
 7. Sound effects of shooting and weapon reloading
 8. Minimap
